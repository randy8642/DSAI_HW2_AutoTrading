## 使用方式
1. 進入專案資料夾\
`cd DSAI_HW2_AutoTrading`
2. 安裝所需套件\
`pip install -r requirements.txt`
3. 執行\
`python trader.py --training ./data/training.csv -- testing ./data/testing.csv --output ./output.csv`
